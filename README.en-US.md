# AulaPos

[Versao em Portugues](README.md)

Academic Java project used for study during a Software Architecture Specialization degree.

## Purpose

This repository is a learning workspace for practicing Java, Spring Boot, persistence, web APIs, and software architecture concepts covered during postgraduate coursework.

The project may evolve over time as new classes, examples, architectural patterns, and experiments are added.

## Stack

- Java 17
- Spring Boot
- Spring Web MVC
- Spring Data JPA
- H2 Database
- Maven

## Requirements

- JDK 17 or newer
- Maven, or the included Maven Wrapper

## Running the Project

Using the Maven Wrapper:

```bash
./mvnw spring-boot:run
```

On Windows:

```powershell
.\mvnw.cmd spring-boot:run
```

## Running Tests

```bash
./mvnw test
```

On Windows:

```powershell
.\mvnw.cmd test
```

## Project Structure

```text
src/
  main/
    java/        Application source code
    resources/   Application configuration and resources
  test/
    java/        Automated tests
```

## Study Goals

- Practice Java application development with Spring Boot
- Explore layered architecture and separation of concerns
- Work with REST controllers and web endpoints
- Study persistence using JPA
- Use tests to validate application behavior
- Document architectural decisions as the project grows

## Notes

This is an academic repository. Code and structure may change frequently as part of experimentation, class exercises, and architecture studies.
