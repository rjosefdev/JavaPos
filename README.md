# AulaPos

[English version](README.en-US.md)

Projeto academico em Java usado para estudo durante uma especializacao em Arquitetura de Software.

## Proposito

Este repositorio e um espaco de aprendizado para praticar Java, Spring Boot, persistencia, APIs web e conceitos de arquitetura de software abordados durante a pos-graduacao.

O projeto pode evoluir ao longo do tempo conforme novas aulas, exemplos, padroes arquiteturais e experimentos forem adicionados.

## Stack

- Java 17
- Spring Boot
- Spring Web MVC
- Spring Data JPA
- H2 Database
- Maven

## Requisitos

- JDK 17 ou superior
- Maven, ou o Maven Wrapper incluido no projeto

## Executando o Projeto

Usando o Maven Wrapper:

```bash
./mvnw spring-boot:run
```

No Windows:

```powershell
.\mvnw.cmd spring-boot:run
```

## Executando os Testes

```bash
./mvnw test
```

No Windows:

```powershell
.\mvnw.cmd test
```

## Estrutura do Projeto

```text
src/
  main/
    java/        Codigo-fonte da aplicacao
    resources/   Configuracoes e recursos da aplicacao
  test/
    java/        Testes automatizados
```

## Objetivos de Estudo

- Praticar desenvolvimento de aplicacoes Java com Spring Boot
- Explorar arquitetura em camadas e separacao de responsabilidades
- Trabalhar com controllers REST e endpoints web
- Estudar persistencia usando JPA
- Usar testes para validar o comportamento da aplicacao
- Documentar decisoes arquiteturais conforme o projeto evolui

## Observacoes

Este e um repositorio academico. O codigo e a estrutura podem mudar com frequencia como parte de experimentos, exercicios de aula e estudos de arquitetura.
